Key Internet Protocols
  • HTTP

    Foundation of data communication on the Web. Defines how messages are Formatted and transmitted.

  • HTTPS

    Secure version of HTTP using encryption (SSL/TLS). Ensures secure data transfer, especilly for sensitive information.

  • TCP/IP

    Fundamental suite of protocols for internet communication. Facilitates reliable data transmission between connected devices.

Client-Server Architecture
  • Roles

    Clients request services or resources. Servers provide requested services or resources.

  • Request-Respones Cycle

    Clients send requests to servers. Servers process requests and send back responses.

  • Satelessness

    Each request from a client to a server is independent. No retention of client state between requests.

Internet Technologies
  • HTML

    Standard markup language for creating web pages. Defines structure and layout using tags.

  • CSS

    Style sheet language for description the presentation of a document. Enhance the visual presentation of HTML elements.

  • JavaScript

    Programming language that enables dynamic content on web pages. Enhance interactivity and user experience.
